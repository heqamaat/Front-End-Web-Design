I am using Visual Studio Code to complete this assignment.

To plan a website, it's important to identify the purpose or goal of the site as well as the target audience. Then you can start to develop are rough plan for a site map and sketch out wire frames. These high level plans will allow you to make more granular decisions about site details such as the graphics, color and typography. Finally, consideration should be given to matters of accessibility, budget and the timeline for completing the project.

I thought it was particularly useful to learn about some of the details in Exploring the web. It's important to understand how all the pieces of the network fit together such as the protocols that various devices in the network use to communicate with one another. We'll be making use of these systems regularly so understanding how they work helps us to better utilize them.
